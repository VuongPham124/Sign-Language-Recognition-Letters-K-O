The project focused on developing a real-time sign language recognition system using deep learning. Multiple models, including ResNet50, MobileNetV2, and custom CNN, were trained and evaluated to identify sign language letters with high accuracy. MediaPipe was integrated for precise hand tracking, enabling real-time gesture detection. The complete system was deployed through Streamlit, providing an interactive interface for live testing and visualization.
